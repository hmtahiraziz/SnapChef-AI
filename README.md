# AI Recipe App

Expo project is in [`ai-recipe-app/`](./ai-recipe-app/) because the parent folder name contains spaces (Expo requires URL-safe names).

Backend API is in [`backend/`](./backend/) (FastAPI + MongoDB + OpenAI).

```powershell
cd ai-recipe-app
npm install
copy .env.example .env
npm start
```

See [ai-recipe-app/README.md](./ai-recipe-app/README.md) for full setup and structure.
See [backend/README.md](./backend/README.md) for backend setup and API usage.
